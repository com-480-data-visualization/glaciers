COM-480: Swiss Glaciers

Swiss Glaciers Visualizations

This project is an interactive website aimed to visualize glacier retreat across Switzerland. It focuses on differences in rate of change, not only on static glacier outlines, and enables users to move from a national overview to glacier-specific detail views.
